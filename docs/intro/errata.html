# 99ページ（第2章）

  • スライド最下部

# 誤 chmod ug-wx file.sh

# 正 chmod go-wx file.sh

ug-wxでは、所有者とグループから書き込みと実行権限を削除することになるため、誤りです

# 135ページ（第3章）

  • スライド中央部

# 誤 locate *.tmp|xargs rm

# 正 locate .tmp|xargs rm

.tmpで終わるファイル名（hoge.tmpなど）を検索するときはこれは誤りではありません。ファイル名の一部に .tmp を含むファイル（hoge.tmp.orgなど）をすべて検索したい場合、このコマンドでは期待する結果が得られないので注意してください

# 191ページ（第4章）

  • groupmodの解説

# 誤 groupmodコマンドは、ユーザーをサブグループに追加することもできます

# 正 groupmodコマンドはグループの設定を変更するコマンドなため、ユーザーをグループに参加させることはできません。通常は usermod コマンドを使います

一部のディストリビューション（比較的新しいものなど）では、groupmodコマンドでユーザーをサブグループに所属させることも可能です
